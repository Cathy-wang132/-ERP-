# 进销存管理系统源码

## 一、介绍

进销存管理系统源码，助您轻松掌握企业的采购、库存和销售！


## 二、软件架构

编程语言： JAVA

后端框架： spring-boot

前端组件： element-ui

数据库： mysql5.7

缓存技术： redis
